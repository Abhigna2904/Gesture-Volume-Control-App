# 🎛️ Gesture Volume Controller

Control your system volume using real-time hand gestures through your webcam. This project uses computer vision to detect finger movements and adjust system volume dynamically.

---

## 🚀 Features

* ✋ Real-time hand tracking using MediaPipe
* 🎥 Live webcam feed with OpenCV
* 🔊 Control volume using thumb–index finger distance
* 📊Visual feedback for volume level and finger distance
* 🌙 Simple and clean user interface

---

## 🛠️ Tech Stack

Python • Flask • OpenCV • MediaPipe • NumPy • Pycaw

---

## ⚙️ Installation & Setup

```bash
git clone https://github.com/YOUR-USERNAME/Gesture-Volume-Control-App.git
cd Gesture-Volume-Control-App
pip install -r requirements.txt
python app.py
```

---

## 🎮 How to Use

1. Run the application
2. Click Start Camera
3. Show your hand in front of the webcam
4. Adjust the distance between thumb and index finger
5. System volume changes automatically
